Frequently Asked Questions

Dom stands for Document Object Model. It is a cross-platform and language-independent interface that helps to connect web pages to scripts or programming languages.

There are many ways to select HTML elements using DOM. The most common methods are: getElementsByTagName getElementsByClassName getElementById querySelector querySelectorAl by using these methods we can select HTML elements.

event bubbling is a mechanism in which the event is first captured and handled by the innermost element and then propagated to outer elements.

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.
